What is Windows Vista?

Windows Vista is an operating system (OS) released by Microsoft in late January 2007. The OS featured a new graphical interface, the search component "windows search", redesigned networking, and new display subsystems. Microsoft claimed that the primary goal of Vista was to improve the security of the Windows line of operating systems as a common complaint of the previous generation OS is that had easily exploited security loopholes and was very susceptible to many viruses and buffer overflows. This meant that Vista was primarily an effort to incorporate security in all aspects of the OS.

Reception and criticism

With all of the security incorporated in this iteration of Windows, there was some general positivity with this OS, like the new graphical interface.
However, the OS had been targeted by vast criticism as it had many issues.

Main Points of Criticism

  • The OS had very resource demand and effectively limited many PCs to very minute functions such as E-mail.
  • Licencing was very restrictive compared to previous generations.
  • Inclusion of new digital rights management technology helped to make acts of piracy more difficult
  • Very long boot time compared to previous generations.
  • A visible lack of compatibility with pre-Vista hardware and software.

System management

While much of Vista focused on security and user interface, there were also additions to deployment and maintenance features.

  • Windows image format, these files which contain a hardware abstraction layer image can be maintenanced without rebuilding new images.
  • Windows deployment services, replaced remote installation services for deployment of all Windows operating systems.
  • Multilingual user interface, unlike other iterations of Windows Vista will change languages to match the users preference.
  • This was the first version of Windows to support wireless projector use.
